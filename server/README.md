# KidzPAN Milestone Tracker API

    Backend assignment for KidzPAN — Implements REST APIs to manage Children’s Milestones.


---
## 1. Tech Stack

- Node.js + Express
- PostgreSQL
- Schema and Seeding Data in SQL
- Docker for Containerization
